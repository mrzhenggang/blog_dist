Materials Studio 任务提交脚本详解

Materials Studio是美国Accelrys公司生产的新一代材料计算软件，它是商业软件哦。

在天河系统上如果用户希望使用它，请安装使用正版软件。

结合slurm的作业管理系统，可以使用我们编写的 ms-th-1A.sh 脚本提交任务。

这个脚本的使用方法为：

yhbatch -N NNODES -n NPROCS -p PARTITION ms-th-1A.sh server seedname
1

参数说明：

参数 含义
NNODES 使用的节点数
NPROCES 使用的总核数
PARTITION 计算分区铭（可以用 yhi 命令查看
server 模块名称，如castep, dmol or dftb
seedname xxx.param 前面的xxx，不需要加后缀

举一个例子，如果希望用 2 个节点，24 个CPU核心，TH_SR1 分区，使用castep 模块，计算输入文件为 Al.param 的任务，可以使用如下命令：

yhbatch -N 2 -n 24 -p TH_SR1 ms-th-1A.sh castep Al
1

当你忘了这个脚本怎么用的时候，可以：

chmod +x ./ms-th-1A.sh
./ms-th-1A.sh
1
2

就会显示一下内容：

yhbatch -N NNODES -n NPROCS -p PARTITION ms-th-1A.sh server seedname         
                                                                             
    NNODES    :  number of nodes on which to run                             
    NPROCES   :  number of tasks to run                                      
    PARTITION :  partition requested (use 'yhi' to check available partition)
                                                                             
    server    :  server name (castep, dmol or dftb)                          
    seedname  :  seed name
