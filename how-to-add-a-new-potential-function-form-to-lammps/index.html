如何在lammps里加入新的势函数形式?

# 官方

参考官方手册，先看看：传送门 (opens new window)。

# 自己编程

如果没有，就自己写吧。

可以先参考已经有的势函数，弄清楚框架和写法，再写自己的。有的时候不仅要写 .cpp 文件，还要写一个 .h 头文件。

# 编译

把写好的东东放在lammps/src目录，然后按照以前的方法make即可。

可以参考文章：lammps 编译安装教程 (opens new window)。
