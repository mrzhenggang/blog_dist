R编译安装教程

# 简介

R是用于统计分析、绘图的语言和操作环境。R是属于GNU系统的一个自由、免费、源代码开放的软件，它是一个用于统计计算和统计制图的优秀工具。

官网地址：传送门 (opens new window)。

# 编译

在集群上安装R环境，安装R-3.0.0版本的时候比较顺利，安装最新版本R-3.4.4的时候，由于依赖比较多且软件版本要求比较新，故需要预先安装很多包，否则报错。

我走通流程后将其记录如下：

# 设置安装目录
R_INSTALL_DIR=$HOME/software/R-3.4.4
export PATH=$R_INSTALL_DIR/bin:$PATH

# zlib
tar zxvf zlib-1.2.11.tar.gz
cd zlib-1.2.11
./configure --prefix=$R_INSTALL_DIR
make -j 12
make install
cd ..

# bzip
tar zxvf bzip2-1.0.6.tar.gz 
cd bzip2-1.0.6
make -j 12
make install PREFIX=$R_INSTALL_DIR
cd ..

# xz
tar zxvf xz-5.2.3.tar.gz
cd xz-5.2.3
./configure --prefix=$R_INSTALL_DIR
make -j 12
make install
cd ..

# pcre  "--enable-utf8" 因为提示需要支持这个，而默认是关闭的
tar zxvf pcre-8.42.tar.gz 
cd pcre-8.42
./configure --prefix=$R_INSTALL_DIR --enable-utf8
make -j 12
make install

# curl
tar zxvf curl-7.59.0.tar.gz
cd curl-7.59.0
./configure --prefix=$R_INSTALL_DIR
make -j 12
make install
cd ..

# R
tar zxvf R-3.4.4.tar.gz 
cd R-3.4.4
./configure --prefix=$R_INSTALL_DIR \
  CPPFLAGS="-I$R_INSTALL_DIR/include" \
  LDFLAGS="-L$R_INSTALL_DIR/lib"
make -j 12
make install
cd ..
