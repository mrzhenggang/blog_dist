登录VPN

# 登陆 VPN

目前 TH-1A 系统已经同时接入了联通和电信双网络，用户根据自身网络接入商的不同可以选择不同的登陆域名来登录 VPN。另外，VPN 可支持手机登陆，目前安卓/苹果系统的手机均可登陆 VPN。

# VPN网址

TH-1A系统的VPN网址：

运营商 网址
联通 https://thvpn.nscc-tj.cn
电信 https://thvpn1.nscc-tj.cn

百度搜索，输入 ip 即可获得当前的ip地址和网络运营商。

# 初次登陆VPN

其实，输入网址，再输入用户名和密码就ok了，如果提示下载安装插件，就允许。做了一个小动画分享给大家。

该方法适用于各类操作系统，如果是Linux64位操作系统，请下载专用VPN脚本install_vpn_linux_x86.sh (opens new window)，提取码：1j8z

手机端也可以登陆VPN，登陆方法可以参考：EasyConnect-移动端登陆.pdf (opens new window) ，提取码：vykc

# 再次登陆VPN

可以通过浏览器，输入网址，再输入用户名和密码即可。和初次登陆不同，这一次不需要安装插件，因为第一次已经安装好了。

我们更推荐下面这种：

通过开始菜单，搜索并打开"Easy Connect"（通常输入easy就够了），输入用户名和密码，点击登陆即可。

# VPN修复

如果遇到VPN无法登陆的情况，昨天还好好的，今天就不行了，我啥也没改啊，怎么办？？

推荐使用修复工具：

SSL VPN Windows环境诊断修复工具，一键检测电脑插件问题 (opens new window)

详细介绍可以参考：http://mrzhenggang.com/vpn-sangforhelpertoolinstaller/

# 注意事项

提示

  1. 在国家超级计算天津中心网站 www.nscc-tj.cn 右上角有"用户 VPN登陆"选项，用户也可以从那里点击进入（联通与电信用户请分别选择适合自己的网络接入）。
  2. 如果您在 Windows 下安装有 360 安全卫士等防护软件，当有提示时，请点允许，来保证 VPN 的正常使用。
  3. 由于 VPN 账号同时允许多人登陆，因此不允许修改密码，您如果需要修改密码，请联系中心技术人员。
  4. 系统默认用户登陆 VPN 后，如不进行任何操作（登录系统等操作），30 分钟后会自动下线。
