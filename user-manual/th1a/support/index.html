# 技术支持

# 官方

由于用户手册篇幅有限，只列出了用户使用系统的基本方法以及常见问题和解决方法，很难面面俱到，还请您能够谅解。如果您在系 统使用过程中遇到任何问题，都可以及时与中心技术人员取得联系。

  1. 合同、资源申请使用、应用软件相关问题联系方式：
  • Email：service@nscc-tj.cn
  • 电话：022-65375561
  1. 系统使用、作业运行相关问题联系方式：
  • Email：support@nscc-tj.cn
  • 电话：022-65375560

重点提示：为了能尽快使您的问题得到定位解决，请您在通过邮件或电话联系中心技术人员时提供以下基本信息（建议拷贝表格填写）：

系统用户名 作业jobid 作业日志路径 作业提交命令 错误现象

包括：系统用户名，出错作业号（jobid），出错作业日志路径，作业提交命令，错误现象描述等信息。

在此，特别感谢您对我们工作的信任与支持，同时也祝您在天河系统使用过程中工作愉快！希望我们能够共同携手推动我国并行计算技术的发展，合理使用并行计算资源在各个科研及工业领域不断创新突破！
