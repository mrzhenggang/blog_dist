# 取消作业

如果提交作业后，如果正常结束，那么在slurm-jobid.out文件中并不会有任务的报错信息。

如果任务已经提交，想将它取消，可以使用 yhcancel 命令，步骤如下：

1）确定任务编号

我们用yhq命令查看任务编号，比如为

$ yhq
JOBID    PARTITION  NAME     USER       ST  TIME      NODES NODELIST(REASON)
1454916  debug      sub.sh   zhenggang  R   00:15:00  2     cn[5-6]
1
2
3

2）取消任务

$ yhcancel 1454916
1
