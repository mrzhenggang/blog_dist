MOLPRO安装使用说明

# 简介

MOLPRO是国际上广泛使用的专业级电子结构量化计算软件，由H.-J. Werner和P. J. Knowles负责设计和维护。不同于其它的量子化学软件包，MOLPRO的重点是高精度计算，通过多参考CI，耦合簇和有关的方法，广泛处理电子相关问题。使用最近开发的直接积分局域电子相关方法，可以极大地减少随分子尺寸增加的计算量，能够对更大的分子体系进行准确的从头计算。程序的核心是MCSCF，MR-CI，和耦合簇程序，以及一系列附加功能。 主页可以申请试用版。

# 下载

官方下载地址 (opens new window)

# 安装

将下载到预先编译的脚本文件molpro-mpp-2015.1.20.ompi.sh上传到系统上，添加可执行权限，直接执行即可安装。

# 创建目录（如果不存在则创建，否则安装过程报错找不到）

mkdir -p $HOME/software/molpro/2015/bin
1

# 添加脚本可执行权限

chmod +x molpro-mpp-2015.1.20.ompi.sh # 下载到预先编译好的脚本文件
1

# 执行脚本

./molpro-mpp-2015.1.20.ompi.sh

# 配置Token

配置Token可以在对应的lib文件夹添加.token文件，例如：

cd $HOME/software/molpro/2015/molprop_2015_1_linux_x86_64_i8/lib
vim .token
	id=***,date=:2025/02/05,version=:2015,mpp=32767,modules=mpp,password=******
1
2
3

然后写入你的账户密码信息，保存即可。 也可以声明环境变量，如：

export MOLPRO_KEY="id=***,date=:2025/02/05,version=:2015,\
    mpp=32767,modules=mpp,password=******"
1
2

# 额外配置*

在实际测试过程中，发现有时候会缺少lscpu命令，所以我们从登陆节点拷贝一份到bin目录：

cp /usr/bin/lscpu $HOME/software/molpro/2015/molprop_2015_1_linux_x86_64_i8/bin
1

# 默认提交

# 默认输出文件明.out
molpro h2o.inp 
# 不保存已有.out文件，而是重写
molpro -s h2o.inp
# 指定输出文件名
molpro -o water.output h2o.inp 
# 设置使用内存
molpro -m 4M h2o.inp
# 8核并行
molpro -n 8 h2o.inp
# 设置临时目录，默认为/tmp/$USER;设置TMPDIR变量，或-d 参数指定
molpro -d /scratch/$USER h2o.inp
